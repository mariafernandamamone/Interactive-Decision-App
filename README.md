🧾 Project Description

This project is a browser-based JavaScript application that interacts with the user through prompts and confirmation dialogs. It includes multiple mini-features such as authentication, car maintenance checks, blood donation eligibility, a quiz, and a rating system.

The goal of this project is to practice fundamental JavaScript concepts, especially logical operators and conditional statements, by building small interactive scenarios that respond dynamically to user input.

📌 Features

🔐 Authentication system (username and password validation)
🚗 Car maintenance check (yes/no conditions)
🩸 Blood donation eligibility test
🎯 Yes/No quiz
⭐ Rating system (1–10 scale with different responses)

🧠 Concepts Used

Logical operators (&&, ||, !)
Comparison operators (===, >=, etc.)
Conditional statements (if, else if, else)
User input with prompt() and confirm()
Basic input validation

▶️ How to Run

Open the index.html file in your browser and follow the on-screen prompts.
